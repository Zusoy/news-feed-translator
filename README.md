# Feed translator

This application can be use to translate news feeds with Deepl API

## Requirements

- Docker 24.0.2+
- docker-compose 1.29+
- Git

## Services

- Deepl API
- Livesquawk API

## Installation

1. clone the repository
2. Copy the `.env.dist` file into `.env`
3. Run

```
make build
```

```
make start
```
